# 🤖 AI Meeting Automation System

## 🚀 Overview
This project is an agentic AI system that automates meeting scheduling using multiple agents.

## 🧠 Features
- Admin Agent sends meeting invitations
- Buyer Agents simulate user responses
- Coordinator Agent decides optimal meeting time
- Automated email confirmations
- Minutes of Meeting (MoM) generation
- Professional UI using Flask

## ⚙️ Tech Stack
- Python (Flask)
- SMTP (Gmail)
- HTML/CSS

## 🔁 Workflow
1. Send meeting invites
2. Collect responses (simulated)
3. Decide best time
4. Send confirmation
5. Send meeting summary

## 🔐 Security
- Uses environment variables for credentials
- Gmail App Password authentication
